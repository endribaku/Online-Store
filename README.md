# Online Store

A simple C#/.NET project that demonstrates an online store application, structured into a front-end (or UI) project and a class library for business logic.

## Overview

This solution consists of a front-end (UI) project and a class library that handles the core business logic and data access. The idea is to separate concerns: the UI handles user interaction, while the class library handles product management, orders, and any store rules.

## Features

- Product listing and management  
- Shopping cart / order processing  
- Separation of concerns: UI layer vs. business logic  
- Intended as a learning / demonstration project

## Usage

1. **Browse Products**  
   Users can view a list of available products in the store interface.

2. **Add to Cart**  
   Select products and add them to a virtual shopping cart.

3. **Checkout**  
   Review the shopping cart and proceed to place an order.

4. **Manage Products / Orders**   
   If admin functionality is enabled, administrators can manage product listings and view customer orders.
